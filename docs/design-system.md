# NEXOLAB Design System

## Візуальний напрям

Modern Industrial Tech UI: темні cold-tech поверхні, синьо-бірюзові сигнали, модульна геометрія та стриманий glow.

## Основні токени

| Роль         | Значення  |
| ------------ | --------- |
| Background   | `#06142A` |
| Deep navy    | `#0B1D3A` |
| Steel blue   | `#132E5F` |
| Primary      | `#0077FF` |
| Cyan         | `#00C6E0` |
| Success      | `#22C55E` |
| Warning      | `#F5B301` |
| Danger       | `#FF4D4F` |
| Text primary | `#E6ECF2` |

## Правила компонентів

- Radius панелей: 16–20 px.
- Border має бути помітним при hover, але не домінувати у normal state.
- Critical state використовує червоний акцент локально, без суцільного червоного заливу.
- KPI value має найвищий контраст усередині картки.
- Графіки використовують інтерактивні legend controls і синхронізовану часову вісь.
- Мінімальна touch target зона інтерактивного елемента — приблизно 40 px.

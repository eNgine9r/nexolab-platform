# Code of Conduct

Учасники проєкту мають працювати професійно, аргументувати технічні рішення та поважати інших учасників. Персональні атаки, дискримінація, переслідування та навмисне зривання роботи не допускаються.

Проблемні ситуації повідомляються власнику репозиторію приватно. Власник може редагувати або видаляти неприйнятний контент та обмежувати доступ порушників.
